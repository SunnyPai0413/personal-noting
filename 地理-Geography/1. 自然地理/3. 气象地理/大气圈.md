## 大气
1. 大气垂直分布：
	1. 对流层（0-12km）：高度随纬度递增而递减，高纬度只有8-9km，上冷下热，对流显著，天气复杂多变
	2. 平流层（12km-50km）：臭氧吸收紫外线，上热下冷，空气水平流动，适合飞行
	3. 高层大气（50km-3000km）：先下降再上升，在紫外线和宇宙射线作用下，有若干电离层，对无线电通信有重要作用
2. 低层大气组成：水汽、干洁空气、固体杂质
3. 干洁空气组成：
基本不变气体：

 | 成分        | 体积分数$\varphi$ | 质量分数$\omega$ | 主要作用                           |
 | ----------- | ----------------- | ---------------- | ---------------------------------- |
 | 氮气($N_2$) | 78.08%            | 75.52%           | 生物体基本成分，稳定大气性状       |
 | 氧气($O_2$) | 20.94%            | 23.15%           | 一切生物维持生命活动不可缺少的物质 |
 | 氩气($Ar$)  | 0.93%             | 1.28%            |                                  |

## 逆温
1. 定义：对流层中气温随高度增加而上升的现象
2. 逆温类型：
	1. 辐射逆温：
		成因：大气逆辐射较弱，地面辐射散失热量多；近地面气温迅速下降，高处大气层降温幅度较小
		特点：常年均可能出现、冬季最强、黎明前最明显，日出后自下而上逐渐消失
	 	![[地理-Geography/1. 自然地理/3. 气象地理/attachments/radiation-inversion.png|550]]
	2. 平流逆温：
		成因：暖空气水平移动到冷的地面或气层上，暖空气的下层迅速降温，上层降温较慢
		特点：多出现在中纬度沿海地区，白天弱夜间强；海面暖湿空气冬季流到中纬度大陆，会出现“平流雾”
  		![[地理-Geography/1. 自然地理/3. 气象地理/attachments/advection-inversion.png|400]]![[地理-Geography/1. 自然地理/3. 气象地理/attachments/advection-fog.png|400]]
	3. 锋面逆温：
		成因：锋面上下冷暖空气的温度差异。暖锋在上冷锋在下，冷暖气团之间过渡带易形成逆温
		特点：只能在冷气团控制的地区观测到、多发于暖锋过境时
		![[地理-Geography/1. 自然地理/3. 气象地理/attachments/frontal-inversion.png|350]]
	4. 地形逆温：
		成因：山坡散热快，冷空气循山坡下沉到谷底，谷底原来的较暖空气被冷空气抬挤上升，从而出现气温的倒置现象
		特点：发生在山谷和盆地晚上
	 	![[地理-Geography/1. 自然地理/3. 气象地理/attachments/topography-inversion.png|350]]
		成因2：冬半年冷空气在向低纬度地区运动过程中，因冷空气较冷重，把地势较低盆地和谷地地区填满（形成冷空气湖），而盆地上空是暖空气
		特点2：发生在冬半年冷锋过境时
	 	![[地理-Geography/1. 自然地理/3. 气象地理/attachments/terrain-inversion.png|550]]![[地理-Geography/1. 自然地理/3. 气象地理/attachments/terrain-inversion-2.png|550]]
	5. 湍流逆温（乱流逆温）：
		成因：低层空气湍流混合作用形成的逆温。上升空气降温、下沉空气升温，即湍流将上层热量搬运到下层（湍流混合层），而上层不变。
		![[地理-Geography/1. 自然地理/3. 气象地理/attachments/turbulent-inversion.png|350]]
		湍流：上层空气温度低于下层导致的空气不规律的流动
		![[地理-Geography/1. 自然地理/3. 气象地理/attachments/atmospheric-turbulence.png|500]]
3. 逆温的影响：
	1. 有利方面：
		1. 逆温的出现阻碍了空气对流，因此可以抑制沙尘暴的发生
		2. 高空的平流逆温对飞机的飞行极为有利，使飞机在飞行中不会有大的颠簸；同时也提高了大气能见度，使飞行更加安全
		3. 和其他天气现象一样，逆温可当成一种气候资源加以利用。例如，在我国新疆伊犁河谷，逆温出现在10月至次年3月，长达半年之久，有效地提高了冬季谷地的温度，多年生果树越冬可以免受冻害。
	2. 不利方面：
		1. 逆温时大气结构比较稳定，容易加重大气污染
		2. 对天气的影响：容易产生大雾等不利天气
	3. 对交通的影响：能见度降低，地面湿滑

## 风
1. 形成原因：
	1. 直接原因：同一水平面上的气压差异产生了水平气压梯度力
	2. 根本原因：冷热不均
2. 影响风的力：
	1. 水平气压梯度力：方向与等压线垂直，高压指向低压，风产生的原因；决定风速，影响风向
	2. 地转偏向力：方向与风向垂直，北半球向右偏（顺时针）、南半球向左偏（逆时针）；不影响风速，改变风向
	3. 摩擦力：方向始终与风向相反；降低风速，影响风向
3. 风向：
	1. 高空风：
	 ![[地理-Geography/1. 自然地理/3. 气象地理/attachments/wind-direction.png|500]]
	  	受水平气压梯度力和地转偏向力的影响，风向最终平行于等压线（平行后水平气压梯度力和地转偏向力平衡）
	2. 近地面风：
   	 ![[地理-Geography/1. 自然地理/3. 气象地理/attachments/wind-direction-2.png|500]]
		受水平气压梯度力、地转偏向力和摩擦力共同作用，风向斜穿过等压线

## 大气环流：
1. 单圈环流：
	1. 定义：假设地球没有地转偏向力、摩擦力等，仅凭水平气压梯度力形成的在赤道与极地之间的闭合环流（实际不存在）
	2. 示意图：
	  ![[地理-Geography/1. 自然地理/3. 气象地理/attachments/single-loop-circulation.png|600]]
2. 三圈环流：
	1. 定义：受太阳加热不均和地转偏向力影响下所形成的环流圈，大气环流的理想模式
	2. 示意图：
		![[地理-Geography/1. 自然地理/3. 气象地理/attachments/three-cell-circulation.png]]
	3. 分类：
		1. 低纬环流：
			赤道气温较高，气压较低，气流膨胀上升；高空气压高，开始向两极运动；受地转偏向力影响，30°附近堆积下沉，使得地表气压高；赤道地表气压低，于是开始向赤道流动
		2. 中高纬环流：
			副热带气压较高，开始向极地运动，受地转偏向力影响最后变为西风气流；极地地区气压较低，空气下沉，向赤道运动，受地转偏向力影响最后变为西风气流；西风气流和东风气流在60°附近相遇，形成极锋。
			来自副热带的西风暖气团爬升到东风上空，形成副极地上升气流；气流上升后使近地面气压降低，形成副极地低压带；上升气流在高空又分成南北两个方向，在副热带和极地分别下沉；在副热带和副极地之间形成中纬环流，极地和副极地之间形成高纬环流。
3. 七带六风：
	1. 七带：赤道低气压带，南北半球的副热带高气压带，南北半球的副极地低气压带，南北半球的极地高气压带
	2. 六风：低纬度的信风、中纬度的西风、极地的东风
	3. 示意图：
	   	![[地理-Geography/1. 自然地理/3. 气象地理/attachments/atmospheric-pressure-and-wind.png|750]]
	4. 季节性变化：
		![[地理-Geography/1. 自然地理/3. 气象地理/attachments/movements-of-barometric-zones-and-wind-belts.png|750]]
		1. 赤道低压带：北半球夏季偏北，南半球夏季偏南
		2. 30°~40°：冬季盛行西风，夏季被副热带高压控制
		3. 偏移幅度一般为5°~10°
4. 季风环流
	1. 成因：下垫面对气压带的破坏
	2. 下垫面：包括地形地势（海拔高低、迎风背风）、海陆位置、洋流、植被覆盖等
	3. 下垫面对局部气候影响：例如“城市五岛”
		1. 热岛效应：
		 	![[地理-Geography/1. 自然地理/3. 气象地理/attachments/heat-island-effect.png|700]]
		   	成因：大量的人工发热、钢筋混凝土建筑高蓄热、绿地减少，导致城市气温升高，和周边郊区相比就好像突出海面的岛屿
		2. 雨岛效应：
		 	![[地理-Geography/1. 自然地理/3. 气象地理/attachments/suburban-wind.png|700]]
		   	成因：局部辐合上升的气流利于对流雨的发展；城市下垫面粗糙大，对降雨系统移动有迟滞作用，延长降雨时间；城市污染多，凝结核多，有利于形成降水
		3. 干岛效应：
		   	![[地理-Geography/1. 自然地理/3. 气象地理/attachments/hydrographic-features-affected-by-underlying-surface.png|700]]
			成因：下垫面硬化导致地表丧失海绵效应，降雨后由管道快速排至他处；受热岛效应影响，剩余少量水分快速蒸发后，由于城市下垫面缺少土壤和植被的吸收和保蓄能力，难以获得持续的水分补给。所以，城市空气中的水分偏少，湿度较低，形成孤立于周围地区的“干岛”。
		4. 湿岛效应：
			凝露湿岛：天气稳定又无低云、风速较小的夜间，郊区降温快，结露多，空气中水汽大量析出，水汽压即迅速降低；市区因热岛效应，气温较高，结露量较少，空气中水汽压高于郊区，形成城市湿岛。（城郊风会将城市水汽不断送往郊区）上午温度上升后，露水蒸发，郊区空气中湿度迅速增加，市区转为干岛。
			雨天湿岛：降雨时及停雨后，因市区热岛效应使蒸发量比郊区大，空气中水汽含量比郊区多。
			雾天湿岛：雾滴与周围空气间进行水分交换，市区较暖，饱和水汽压较高，能容纳的水汽量较郊区为多。
			结霜湿岛：城市有强热岛效应时发生，类似凝露湿岛。
			雪天湿岛：类似雨天湿岛，但因气温低、热岛效应小和风速稍大等，其湿岛强度较弱。
		5. 雾岛效应（浑浊岛效应）：
		  	![[地理-Geography/1. 自然地理/3. 气象地理/attachments/fog-island-effect.png|700]]
			成因：城市大气中的污染物质比郊区多，凝结核也多，低空的热力湍流和机械湍流又比较强；城市的日照时数减少，太阳直接辐射大大削弱，能见度也小于郊区
		   6. 解决：海绵城市
			 ![[地理-Geography/1. 自然地理/3. 气象地理/attachments/sponge-city.png]]
			以河、湖、池塘等水系和绿地、花园、可渗透路面等城市配套设施这些“海绵体”下渗、滞蓄、净化、回用，最后剩余部分才径流通过管网、泵站外排，可以维持或恢复城市的海绵功能，缓解城市内涝的压力。
	4. 下垫面对七带六风的破坏：
		 1. 季风环流：
			 ![[地理-Geography/1. 自然地理/3. 气象地理/attachments/monsoon-circulation.png|550]]
			 形成原因：
			 热带：气压带、风带的季节性移动
			 亚热带、温带：海陆热力性质差异
		 2. 气压带：
			 ![[地理-Geography/1. 自然地理/3. 气象地理/attachments/surface-barometric-zones-Jul.png|600]]
			![[地理-Geography/1. 自然地理/3. 气象地理/attachments/surface-barometric-zones-Jan.png|600]]
			   季节性变化的气压带基础上，叠加海陆性质差异
		  3. 风带：
			 ![[地理-Geography/1. 自然地理/3. 气象地理/attachments/July-wind-belt.png|1000]]
